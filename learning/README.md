# Learning Steps (00-03)

Each step is a runnable HTML + JS example.
Start a local static server in the repo root, then open the step folder.

Example:

```bash
python -m http.server 5173
```

Then visit:
- http://localhost:5173/learning/00-bootstrap/
- http://localhost:5173/learning/01-events/
- http://localhost:5173/learning/02-view/
- http://localhost:5173/learning/03-renderloop/
- http://localhost:5173/learning/04-transform/
- http://localhost:5173/learning/05-layer/
- http://localhost:5173/learning/06-tile/
- http://localhost:5173/learning/07-tile-queue/
- http://localhost:5173/learning/08-vector/
- http://localhost:5173/learning/09-interaction/
- http://localhost:5173/learning/10-hit-detect/
- http://localhost:5173/learning/11-cache/
- http://localhost:5173/learning/12-worker/

Notes:
- These are minimal implementations for learning. They are not full OpenLayers.
- Each step includes short comments that point to related files in `src/ol`.
